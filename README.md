# Retail Business Analysis

## Project Overview
Analysis of customer behavior, retention, and sales patterns for a retail business.

## Directory Structure
```
retail-business-analysis/
├── data/
│   ├── CustomersData.xlsx
│   ├── Tax_amount.xlsx
│   ├── Discount_Coupon.csv
│   ├── Marketing_Spend.csv
│   └── Online_Sales.csv
├── notebooks/
│   └── 20251005_Business_Case_vF.ipynb
└── src/
```

## Analysis Includes
- Customer Acquisition Analysis
- Retention Rate Analysis
- Sales Pattern Analysis
- Marketing Impact Study

## Dependencies
- pandas
- numpy
- matplotlib
- seaborn

## Usage
1. Clone the repository
2. Install required dependencies
3. Run the Jupyter notebook in notebooks/ directory
